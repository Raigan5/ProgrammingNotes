MongoDB

Install MongoDB on Windows

References:

  • Youtube tutorial
  • London App Brevewery guide (outdated): follow until step 3 (included).

Steps:

  1. Install mongodb server
  2. Create data directory: C:\data
  3. (Optional) Install mongodb shell
  4. Add directories with the server and shell executables to the Path environment variable
  5. Test installations:
    6. mongod --version
mongosh --version

Use MongoDB

Run server:

mongod

Use the client shell

Run shell:

mongosh

Some useful commands:

  • Show available non-empty databases
    • show dbs
  • Show the current database
    • db
  • Create a new database. We won't see it with show dbs until we add some data.
    • use newDBS
  • Show all collections in our current database
    • show collections
  • CRUD operations.
