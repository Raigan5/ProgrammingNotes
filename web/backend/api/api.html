Authentication levels

  1. No authentication
    • Usually, to avoid abuse, there are a rate limit based on the IP.
  2. Basic authentication
    • Header -> Basic: user+pass encoded in base 64
  3. API key authentication
    • Header -> X-API-key: service_api_key
  4. Token based authentication
    • A token is generated when a user logs in.
    • Header -> Authorization: Bearer generated_token.
