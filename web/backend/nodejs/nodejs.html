Node.js

What is Node.js?

Is a server-side runtime environment that allow us to execute JS outside the browser.

Install Node.js on Windows

  1. Go to the official web.
  2. Choose LTS version.
  3. Install it an restart the computer.
  4. Write node -v on the terminal to confirm that it's installed.

Run it

We can use the command node to execute the interpret and work with it.

We can execute a file with the command node filename.js. Usually, the main file is names index.js

NPM (Node Package Manager)

Init the project

npm init //answer the questions
npm init -y //all answers to yes

Install packages

Go to the official repository to find the packages that we need and install them.

npm i package1 package2...

If our project already has a package.json with the packages, we can simply execute this command:

npm i

Import vs require

We can only use one of them in our projects, by default is require. If we want to use import, we need to add 

"type": "module",

to our package.json.
