    1. Create a github repo and clone it.
    2. Create a web called index.html at the root of your repository.
    3. Upload your changes.
    4. Go to Settings - Pages - select main as your branch and wait some minutes.
    5. There is no need to manually redeploy after pushing more changes to the repository.
