How to use it

  1. Go to the official web. Now we have 2 alternatives:
    • Find package manager and use npm or gem to install it.
    • Find Include via CDN and copy what you need:
      • Style: include link in the head section of your HTML. Take care with the CSS priority.
      • Behaviour: include script in the body section.
  2. Copy elements from the documentation or example and adapt it.
    • Find something that looks like what do you want to do, inspect it and copy the div that contains it.
    • Sometimes, it contains customized CSS that we need to copy or SVG that we need to adapt.

  • container
  • container-sm
  • ...

Sized columns

  • The relative size of an element can be specified indicating the number of columns.
  • Multiple breakpoints can be used to specify the relative size in differents sizes of windows: col-sm-12, col-md-8, col-lg-4
  • The default value of each breakpoint is col and is the same than col-12.
  • When the next element doesn't fit in a row, it's moved to the next row.
