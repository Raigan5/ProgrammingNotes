Layout

CSS Display

  • inline: only uses the space it needs. Ignores width and height properties.
  • block: full width. Doesn't allow more element in the same line.
  • inline-block: only uses the space it needs, but we can set the width and height.
  • none: hide elements

Observation: when an inline or inline-block elements don't have enough space in the same line, they are moved to the next line.

CSS Float

  • We can use float: left|right to allow text wrapping around images.
  • We can use clear: left|right|both to ignore float properties of other elements and jump to the next line.
  • Observation: don't use it as a general layout manager.

Responsive websites

Observations:
  • Use browser tools for developers to test responsiveness
  • Maintain the aspect ratio in images:
    • Set height or width and let the other parameter adjust automatically.
    • Use object-fit: cover to maintain aspect ration when the image size is changing due to window resizing.
  • When some properties collide, the result could be unexpected
    • For example, when you explicitly set the width property of an image, it takes priority over horizontal margins. The margins will provide spacing around the image but won't affect the image width unless the combination of the set width and margins exceeds the available width of the container. In that case, the container might overflow or the image might get scaled down to fit within the container.

Media queries

  • Allow us to define conditional CSS that will be executed only when the conditions are satisfied.
  • Be careful with CSS priority. More general conditions should be placed above more specific ones to avoid overriding them.
  • Some examples:
    • @media(max-width: 800px) {
/**CSS for screens below or equal to 800px*/
}

@media(min-width: 500px) {
/**CSS for screens greater than 500px*/
}

@media (min-width: 500px) and (max-width: 800px) {
    /**CSS for screens greater than 500px and below or equal to 800px*/
}

Layout manager

Observation: layouts can be combined.

  • Flexbox (1D)
  • GridLayout (2D)
  • Bootstrap

Flexbox vs Grid

  • Flexbox is 1-dimensional and Grid is 2-dimensional.
  • Grid allows to overlap items.
