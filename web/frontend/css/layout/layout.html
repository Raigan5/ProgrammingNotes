CSS Display

  • inline: only uses the space it needs. Ignores width and height properties.
  • block: full width. Doesn't allow more element in the same line.
  • inline-block: only uses the space it needs, but we can set the width and height.
  • none: hide elements

Observation: when an inline or inline-block elements don't have enough space in the same line, they are moved to the next line.
