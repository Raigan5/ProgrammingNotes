Basic

  • 1 dimensional
  • There are properties for parents and others for childs.
  • display:flex vs "display:absolute|inline..."
    • They are like 2 different systems
    • When we use flex, default layout properties for each element are ignored.
  • display
    • flex: like block
    • inline-flex: like inline-block

Important concepts

Main vs cross axis

Main axis defines how properties are applied (horizontal vs vertical). The non-choosed axis will be the cross axis

Directions

  • Default value is row. Main-axis is horizontal and cross-axis is vertical.
  • column swaps main and cross-axis

Distribution

Parent properties

  • See flexbox guide

Child properties

  • order
  • align-self

Gaps

There are some properties which allow us to define gaps between child elements:

  • gap, row-gap, colum-gap
  • Some values of align-content
  • Some values of justify-content

Examples

Center something in the middle of the screen

//horizontal
    height: 100vh; /*set the height of viewport height*/
    justify-content: center;
    align-items: center;
    display: flex;
    
//vertical
    height: 100%; /*use this instead of viewport height to avoid layout problems*/
