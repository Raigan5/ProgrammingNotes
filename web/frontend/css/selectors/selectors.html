Selectors

Observations:

  • We can combine the combination of the css selectors listed below
  • We can use Tools for Developers to get the selector.

Basic selectors

  • Element: apply it to the tag specified.
    • tag { CSS }
  • Class: define a class and apply this CSS rules to all elements with this class.
    • .class_name { CSS }
  • Id: define an id and apply this CSS rules only to the element with this ID.
    • #id_name { CSS }
  • Attribute: apply it to the elements with this tag and attribute (first version) or with a certain value of this attribute (second version)
    • tag[attr_name] { CSS }
tag[attr_name="value"] { CSS }
  • Universal: select all.
    • * { CSS }

Combining selectors

  • Group: apply the CSS rules to all selectors specified.
    • selector1, selector2,.. {CSS}
  • Direct child: apply the CSS rules to the direct childs childSelector of the parents parentSelector
    • parentSelector > childSelector {CSS}
  • Descendant: apply the CSS rules to the descendants descendantSelector of the ancestors ancestorSelector
    • parentSelector > childSelector {CSS}
  • chaining: apply the CSS rules to the elements where all selectors are true
    • list_of_selectors_without_space {CSS}
