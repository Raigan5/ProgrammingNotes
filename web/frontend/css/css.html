CSS

General

Observations:

  • Browsers apply default CSS styles
  • Some CSS properties apply default values when we don't specify it, increasing the difficulty to debug our code, for example:
    • border: solid black;
border-width: 20px 10px;

If we change the order, the size is overwritten by "border" default size.

border-width: 20px 10px;
border: solid black;

Definition

  • Inline: single element
    • <body style="background-color: #EAF6F6;">
  • Internal: single document
    • <style>
    body {
        background-color: #EAF6F6;
    }
</style>
  • External: can be applied it to several pages and allows us to separate content and style. We should be careful with overwriting when we use CSS from other people, and put in the last lines the CSS that should overwrite the others.
    • <head>
    <link rel="stylesheet" href="css/styles1.css">
    <link rel="stylesheet" href="css/styles2.css">
</head>

Priority (Cascade)

Multiple styles can be applied to the same element, and the priority will determine which one is chosen. From lower to higher priority, we have:

  1. Position: if the CSS rules are defined in the same place, the priority increases from the top to the bottom.
  2. Specifity: how specific is the selector used
    1. Tag
    2. Class
    3. Value/attribute
    4. Id
  3. Type: where is defined
    1. External
    2. Internal
    3. Inline
  4. Importance: write !important inside the value of a CSS property.

Box model

We can create our own boxes with the tags <div> or <span>

From outside to inside we have the following elements:

  • Margin: space outside the box.
    • margin: size;
  • Border: limit of the box.
    • border: size style color;
  • Padding: space inside the box.
    • padding: size;
  • Content

Position

  • Modes
    • Relative: relative to the default position.
    • Absolute: relative to the nearest positioned ancestor or top left corner of webpage.
      • z-index (default is 0): define the z position in a 3 coordinate system
        • Elements with higher z-index are over elements with lower z-index.
        • When an element is absolute, it's over other elements with the same z-index that are not absolute.
    • Fixed: relative to the top left corner of brower's window.
    • Static (default): at the bottom of the last element.
      • left, top,... properties are ignored.
  • Position vs box model: in box model we have a box B1 and the margin is outside of this box. Imagine that you have another box B2 that contains the margin and the box B1, position its related to this box B2.
