Font

Color

See color.

Font-weight

  • normal or bold
  • ligther (-100) or bolder (+100) relative to the parent.
  • number: between 100 and 900

Font-size

  • 1px: pixel (0.26mm)
  • 1pt: point (0.35mm)
  • 1em: 100% of parent's size. It's preferable to use rem, because em can be confusing, specially if we use external CSS.
  • 1rem: 100% of root's size (same that em but using html tag as reference)
  • Names of font sizes

Font-family

General

  • There are two important font families: serif vs santi serif.
  • It's important to have generic fonts as a backup in case the main fonts are not available.

Google fonts

  1. Go to Google fonts.
  2. Select the fonts that you want.
  3. At the right you'll see the selected fonts, copy the link at the head element. Example:
    4. <link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Work+Sans:wght@300&display=swap" rel="stylesheet">
  4. Copy the css rules at the place where do you want to use it. Example:
    5. font-family: 'Work Sans', sans-serif;
