jQuery

Is JS library created to simplify DOM manipulation with JS. Its popularity has been decreasing.

Include jQuery in our project

  1. Go to the official page
  2. Install it with a package manager or go to the next step to use it via CDN
  3. Go to Google or Microsoft CDN and click on it
  4. Select the script and paste it in our code above any other script that uses it. It's something like this:
    5. <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>
  5. Alternative, copy the script line in the head section and add these lines to our scripts:
    6. $(document).ready(function() {
    //our code
})

DOM

Select

Instead of 
document.querySelector("h1");
document.querySelectorAll("h1");
we can do 
jQuery("h1");
or just 
$("h1");
There is no difference in jQuery between selecting one or many.

Traversing

Traversing allow us to find elements based in the relationship with others elements, and this is really useful. For example, given a list of div that contains a check and a paragraph, we can perfom an action over the paragraph related to the check that receives and event.

Manipulate

Styles

change value 
$("h1").css("color", "green");
get the value 
$("h1").css("color");
add class 
$("h1").addClass("myClass myClass2...");
remove class 
$("h1").removeClass("myClass myClass2...");
has class 
$("h1").hasClass("myClass");

Text

add text 
$("h1").text("new text");
add html 
$("h1").html("new html");

Attributes (classes are attributes too)

get the value 
$("h1").attr("attr1");
$("h1").attr("class"); //get all classes
set the value 
$("h1").attr("attr1", "val1");

Elements

create element before other 
$("h1").before("<button>New</button>");
create element after other 
$("h1").after("<button>New</button>");
create element inside and put it at the beginning 
$("h1").prepend("<button>New</button>");
create element inside and put it at the end 
$("h1").append("<button>New</button>");
remove 
$("h1").remove();

Events

$("button").click(function() {
    $("button").css("color", "purple");
});
$("h1").on("mouseover", function() {
    $("h1").css("color", "purple");
});
